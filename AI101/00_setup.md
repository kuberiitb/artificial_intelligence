## Get Groq Key
- If you don't have a credit card, you can't get OpenAI key easiely.
- But you can try Llama models on GROQ for free(for small usage)

### STEPS:
- Signup here [groq/home](https://console.groq.com/home)
- Go to [groq/keys](https://console.groq.com/keys)
- Click on "**Create API Key** and create a new key with any **Display Name**
- Copy the API Key and save it in a notepad file on your computer as **.env** file
- ```GROQ_API_KEY=<KEY YOU COPIED>```

### NEXT:
- Now You are setup with the key.
- Go to 01_test_api_key.ipynb to test your key
